# general-app

#### 介绍
bugu-cli使用模板(管理后台)

#### 软件架构
使用umi3生成基本模板，添加了request, models, cache等模块，开箱即用 🎉

#### 安装教程

1.  mkdir general-app && cd general-app
2.  npm install bugu-cli -g 
3.  bugu-cli

#### 使用说明

1.  npm install
2.  npm start
3.  npm run build
